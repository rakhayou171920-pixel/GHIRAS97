# نادي غِراس - نظام إدارة نقاط الطلاب
## Product Requirements Document (PRD)

### المشكلة الأصلية
بناء تطبيق ويب لإدارة نظام مكافآت الطلاب لنادي غِراس.

---

## المتطلبات المنجزة

### 1. نظام تسجيل الدخول
- مشرف رئيسي واحد (ghiras2026 / ghras2026)
- JWT token مع صلاحية 30 يوم
- حماية كل عمليات الكتابة بالمصادقة

### 2. إدارة الطلاب
- إضافة/تعديل/حذف الطلاب مع صور
- تجميع حسب المجموعة (المشرف)
- رفع صور من الجهاز

### 3. نظام النقاط
- نقاط إيجابية: حضور +10، مشروع +20، إلقاء كلمة +20
- نقاط سلبية: تأخير -5، غياب -40، تلفظ -40
- نقاط مخصصة مع سبب
- **نقاط جماعية**: إضافة نقاط لكل أفراد المجموعة مرة واحدة مع سبب
- **سجل النقاط**: تتبع كل تغيير في النقاط مع التاريخ والسبب

### 4. المهام الأسبوعية
- المشرف ينشئ مهام لكل مجموعة مع نقاط
- تظهر المهام لجميع طلاب المجموعة
- الطالب يحجز المهمة (تختفي من باقي الطلاب)
- المشرف يعتمد المهمة بعد التأكد من الإنجاز
- الحذف التلقائي بعد أسبوع

### 5. الدوري الكروي
- كل مجموعة = فريق كرة قدم
- المشرف يدخل نتائج المباريات يدوياً
- جدول ترتيب تلقائي (نقاط، فوز، تعادل، خسارة، أهداف)
- يظهر في الداشبورد وصفحة الطالب

### 6. نجم الدوري
- يُضاف يدوياً من المشرف مع سبب
- يظهر كشريط في الداشبورد وصفحة الطالب
- سجل لتاريخ النجوم السابقين

### 7. صفحة الطالب العامة
- الترتيب: نجم الدوري → الترتيب والنقاط → جدول الدوري → مسابقة رمضان → المنافسات → المهام الأسبوعية → سجل النقاط
- QR Code للوصول السريع

### 8. روابط المشاهدة للمشرفين الفرعيين
- المشرف الرئيسي ينشئ روابط مع اسم المشرف الفرعي
- الرابط يعطي حق المشاهدة فقط (بدون تعديل)
- يظهر اسم المشرف الفرعي في الصفحة

### 9. المنافسات/التحديات
- أسئلة اختيار من متعدد مع نقاط
- تفعيل/تعطيل المنافسات

### 10. مسابقة رمضان
- 15 سؤال يومي (تبدأ 16 فبراير 2026)
- 20 نقطة لكل إجابة صحيحة

### 11. شريط المتصدرين
- بودويم لأفضل 3 طلاب
- شريط أفقي قابل للسحب

## البنية التقنية

### Backend: FastAPI + MongoDB
- server.py: كل الـ API endpoints
- JWT auth مع PyJWT

### Frontend: React + Tailwind CSS
- LoginPage.js: تسجيل الدخول
- DashboardNew.js: لوحة التحكم (أقسام: طلاب، مهام، دوري، نجم، روابط)
- StudentProfilePublic.js: صفحة الطالب العامة
- FootballLeague.js: إدارة الدوري
- TasksManager.js: إدارة المهام
- LeagueStarManager.js: نجم الدوري
- ViewerLinksManager.js: روابط المشاهدة
- ViewerPage.js: صفحة المشاهدة
- PointsModal.js: نافذة النقاط
- LeaderboardTicker.js: شريط المتصدرين
- RamadanQuiz.js: مسابقة رمضان
- ChallengesManager.js: إدارة المنافسات

### قاعدة البيانات
- students, challenges, tasks, matches, league_star, points_log, viewer_links
